### System과 User 2가지만 사용하는 예시

  
system과 user 프롬프트를 함께 사용하여, 두 역할의 차이점이 드러나도록 설정한 다양한 예시입니다.  
  

### 1. 예시: 친절하고 전문적인 답변 제공

  
System Prompt:

```python
당신은 명확하고 자세하며 전문적인 답변을 제공하는 유용한 어시스턴트입니다.
복잡하거나 기술적인 주제에도 항상 친근한 어조를 유지하십시오.
```

  
User Prompt:

```python
기계 학습의 개념을 쉽게 설명해 줄 수 있나요?
```

  
설명:

- System Prompt는 모델이 친절하고 전문적인 태도로 답변을 제공하도록 설정합니다.
- User Prompt는 사용자가 요청하는 질문을 나타내며, 모델은 시스템의 지시에 따라 복잡한 개념을 쉽게 설명하면서도 친근한 어조를 유지합니다.

  
  

### 2. 예시: 창의적인 글쓰기 요청

  
System Prompt:

```python
당신은 시, 이야기 또는 예술적인 콘텐츠를 창의적이고 상상력 넘치게 생성할 수 있는 AI입니다.
```

  
User Prompt:

```python
겨울 아침의 아름다움을 주제로 짧은 시를 써 주세요.
```

  
설명:

- System Prompt는 모델이 창의적이고 예술적인 작업을 잘 수행할 수 있도록 설정합니다.
- User Prompt는 구체적인 작업을 요청하는 부분으로, 모델은 시스템의 지침에 따라 감성적이고 창의적인 시를 작성합니다.

  
  

### 3. 예시: 문제 해결과 분석적 접근

  
System Prompt:

```python
당신은 복잡한 문제를 분석하고 단계별로 해결책을 제시하는 논리적이고 분석적인 어시스턴트입니다.
```

  
User Prompt:

```python
수학 문제입니다: 사과가 3개 있고, 5개를 더 사면 총 몇 개의 사과가 되나요?
```

  
설명:

- System Prompt는 모델이 논리적이고 분석적인 방식으로 문제를 해결할 수 있도록 설정합니다.
- User Prompt는 수학 문제를 제시하며, 모델은 이를 단계별로 분석하고 계산하여 답변을 제시합니다.

  
  

### 4. 예시: 기술적인 설명 제공

  
System Prompt:

```python
당신은 컴퓨터 과학의 전문가이며, 복잡한 기술적 개념을 중요한 세부사항을 잃지 않으면서도
쉽게 이해할 수 있도록 설명합니다.
```

  
User Prompt:

```python
프로그래밍에서 스택과 큐의 차이점은 무엇인가요?
```

  
설명:

- System Prompt는 모델이 기술적인 개념을 쉽게 설명하도록 설정합니다.
- User Prompt는 구체적인 프로그래밍 개념을 묻는 질문으로, 모델은 시스템의 지시에 따라 이 두 자료 구조의 차이점을 명확하고 간단하게 설명합니다.

  
  

### 5. 예시: 친절한 고객 지원

  
System Prompt:

```python
당신은 항상 공감하고 인내심을 가지고 고객의 문의에 응답하는 고객 지원 어시스턴트입니다.
상황이 어렵더라도 항상 친절하게 대응하십시오.
```

  
User Prompt:

```python
손상된 제품을 받았어요. 환불이나 교환이 가능한가요?
```

  
설명:

- System Prompt는 모델이 고객 지원에서 항상 친절하고 공감하는 태도로 대응하도록 설정합니다.
- User Prompt는 고객의 불만을 나타내며, 모델은 시스템 지침에 따라 문제를 해결하기 위한 공감적이고 차분한 대응을 합니다.

  
  

### 6. 예시: 요약 및 간단한 설명

  
System Prompt:

```python
당신은 긴 문단이나 기사를 핵심적인 의미를 잃지 않으면서 간단하고 이해하기 쉽게 요약하는 전문가입니다.
```

  
User Prompt:

```python
"Moby Dick"의 줄거리를 3문장으로 요약해 주세요.
```

  
설명:

- System Prompt는 모델이 긴 내용을 요약할 수 있도록 설정합니다.
- User Prompt는 구체적인 요약 요청을 하며, 모델은 시스템 지침을 바탕으로 간결하면서도 핵심적인 요약을 제공합니다.

  
  

### 7. 예시: 학술적이고 깊이 있는 설명

  
System Prompt:

```python
당신은 다양한 학문 분야에 대한 전문 지식을 가진 학술적 어시스턴트입니다.
자세하고 정확한 답변을 제공합니다.
```

  
User Prompt:

```python
상대성 이론의 주요 원리를 간단히 설명해 주세요.
```

  
설명:

- System Prompt는 모델이 학술적이고 깊이 있는 정보를 제공할 수 있도록 설정합니다.
- User Prompt는 복잡한 주제를 간단히 설명하되, 그 핵심 원리를 쉽게 전달할 수 있도록 요구합니다.

  
  

### 8. 예시: 비즈니스 및 전략적 조언

  
System Prompt:

```python
당신은 현재 시장 동향과 데이터를 바탕으로 실용적이고 전략적인 추천을 제공하는 비즈니스 어드바이저입니다.
```

  
User Prompt:

```python
내 작은 전자상거래 비즈니스를 향후 6개월 동안 성장시키기 위해 어떤 단계들을 밟아야 할까요?
```

  
설명:

- System Prompt는 모델이 비즈니스 및 전략적 관점에서 유용한 조언을 제공하도록 설정합니다.
- User Prompt는 구체적인 비즈니스 성장 전략을 묻는 질문으로, 모델은 최신 시장 동향을 반영한 실용적인 답변을 제공합니다.
