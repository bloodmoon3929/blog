### system, user, assistant를 모두 사용하는 예시

  
프롬프트 엔지니어링에서 system, user, assistant를 모두 사용하는 예시는 각 역할이 어떻게 다르게 응답하는지 보여주는 좋은 예시가 될 수 있습니다.  
각 역할의 차이를 잘 드러내기 위해 system은 모델의 행동을 정의하고, user는 질문을 제공하며, assistant는 실제로 응답을 합니다.  
아래는 그 예시입니다.  
  

### 1. 예시: 고객 지원 시나리오

  
System Prompt:

```python
당신은 친절하고 공감 능력이 뛰어난 고객 지원 어시스턴트입니다.
고객의 불만이나 질문에 대해 항상 차분하고 전문적인 태도로 응답하며, 문제 해결을 위한 최선의 방법을 제시합니다.
```

  
User Prompt:

```python
제 주문이 잘못 배송되었어요. 어떻게 해결할 수 있나요?
```

  
Assistant Response:

```python
고객님, 불편을 끼쳐드려 죄송합니다.
잘못 배송된 제품에 대해 환불 또는 교환 절차를 진행해 드릴 수 있습니다.
주문 번호와 잘못 배송된 제품을 알려주시면, 신속히 처리해 드리겠습니다.
추가로 궁금한 점이 있으시면 언제든지 문의해 주세요.
```

  
설명:

- System Prompt는 모델의 응답 스타일을 정의하며, 고객 지원을 친절하고 전문적으로 처리하도록 지시합니다.
- User Prompt는 사용자가 직면한 문제를 설명합니다.
- Assistant Response는 모델이 사용자의 요구를 충족시키기 위해 친절하고 공감적인 응답을 제공합니다.

  
  

### 2. 예시: 기술적인 질문에 대한 답변

  
System Prompt:

```python
당신은 컴퓨터 과학의 전문가입니다.
복잡한 기술적 문제를 쉽게 설명할 수 있도록 노력하며, 중요한 세부사항을 놓치지 않습니다.
```

  
User Prompt:

```python
클라우드 컴퓨팅이란 무엇인가요? 쉽게 설명해주세요.
```

  
Assistant Response:

```python
클라우드 컴퓨팅은 인터넷을 통해 데이터를 저장하고 처리할 수 있는 기술입니다.
즉, 컴퓨터나 서버가 아닌, 인터넷 상의 서버에서 데이터를 처리하거나 저장하는 방식입니다.
예를 들어, 구글 드라이브나 아마존 웹 서비스(AWS)가 클라우드 컴퓨팅의 일종입니다.
이를 통해 언제 어디서나 데이터를 접근하거나 관리할 수 있습니다.
```

  
설명:

- System Prompt는 모델이 기술적인 개념을 쉽게 설명하도록 지시합니다.
- User Prompt는 구체적인 기술적 개념에 대한 질문을 제공합니다.
- Assistant Response는 모델이 질문을 쉽게 이해할 수 있도록 간단하고 명확하게 설명하는 방식으로 응답합니다.

  
  

### 3. 예시: 창의적인 글쓰기 요청

  
System Prompt:

```python
당신은 창의적이고 상상력 넘치는 작가입니다. 시, 이야기, 또는 창작물에 대해 요청을 받으면,
감성적이고 독특한 스타일로 응답합니다.
```

  
User Prompt:

```python
가을의 아름다움을 주제로 한 짧은 시를 써주세요.
```

  
Assistant Response:

```python
가을의 바람 속에서  
낙엽은 춤을 추고,  
황금빛 햇살은 부드럽게 내려와  
우리 마음을 따뜻하게 감싸네.  
이 순간, 시간이 멈춘 듯,  
가을은 우리의 마음에 스며들고.
```

  
설명:

- System Prompt는 모델이 창의적이고 감성적인 응답을 생성하도록 설정합니다.
- User Prompt는 창의적인 요청을 하며, 가을의 아름다움을 시로 표현해 달라고 요청합니다.
- Assistant Response는 모델이 이를 창의적이고 감성적인 시로 작성하여 응답합니다.

  
  

### 4. 예시: 수학 문제 해결

  
System Prompt:

```python
당신은 논리적이고 분석적인 어시스턴트입니다.
수학 문제를 해결할 때, 문제를 단계별로 풀어가며 명확하고 정확하게 설명합니다.
```

  
User Prompt:

```python
5x + 3 = 23 문제를 풀어주세요.
```

  
Assistant Response:

```python
먼저, 5x + 3 = 23에서 3을 양변에서 빼줍니다.  
5x = 20  
다음으로, 양변을 5로 나눕니다.  
x = 4  
따라서, x의 값은 4입니다.
```

  
설명:

- System Prompt는 모델이 논리적이고 단계적인 방식으로 문제를 해결하도록 설정합니다.
- User Prompt는 수학 문제를 제시하며, 모델은 이를 단계별로 풀어가며 명확히 설명합니다.
- Assistant Response는 모델이 문제를 풀면서 각 단계를 명확히 설명하는 방식으로 응답합니다.

  
  

### 5. 예시: 비즈니스 전략 조언

  
System Prompt:

```python
당신은 비즈니스 전략과 마케팅에 대한 전문 지식을 갖춘 어시스턴트입니다.
사용자가 요청하는 전략적 조언을 제공할 때, 구체적이고 실행 가능한 방법을 제시합니다.
```

  
User Prompt:

```python
온라인 쇼핑몰을 시작하려고 하는데, 초기에 어떤 마케팅 전략을 쓰면 좋을까요?
```

  
Assistant Response:

```python
온라인 쇼핑몰 초기에 효과적인 마케팅 전략은 소셜 미디어 활용, 검색엔진 최적화(SEO), 그리고 인플루언서 마케팅입니다.  
1. 소셜 미디어 활용: 인스타그램, 페이스북, 유튜브 등에서 쇼핑몰을 홍보하고, 타겟 고객층과의 소통을 강화하세요.  
2. 검색엔진 최적화(SEO): 웹사이트의 콘텐츠와 구조를 최적화하여 구글 검색 결과에서 상위에 노출되도록 하세요.  
3. 인플루언서 마케팅: 인기 있는 인플루언서와 협업하여 제품을 홍보하면 빠른 시간 안에 많은 고객을 유치할 수 있습니다.  
이 전략들을 결합하여 점차적인 고객층을 확보할 수 있습니다.
```

  
설명:

- System Prompt는 모델이 비즈니스 전략에 대해 실용적이고 구체적인 조언을 제공하도록 설정합니다.
- User Prompt는 비즈니스 시작에 대한 질문을 하며, 모델은 이에 대해 실행 가능한 전략을 제시합니다.
- Assistant Response는 모델이 구체적인 마케팅 전략을 제시하며, 각 방법의 효과와 실행 방법을 설명합니다.
