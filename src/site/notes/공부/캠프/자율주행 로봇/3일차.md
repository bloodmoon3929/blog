## 웨이포인트
모의 시험에서 테스트할 모델 다운로드

다운로드가 완료되면 자동으로 연결상태가 됨

계획에서 웨이포인트를 찍고, 데이터에서 armed상태에 자동모드를 하게되면 알아서 움직임

이때 loiter을 사용해 원형으로 돌거나
Return to launch를 통해 주행이 끝날시 집결지점으로 가게하거나
Do set servo를 통한 서보모터 제어
등이 가능함
