1. node.js 설치
2. 터미널을 열고,
	`npx create-next-app@latest (원하는 이름)`
	추가로 설정들이 나오지만, 새롭게 만들 것이니 무시해도 무방
3. vscode로 실행
4. [bloodmoon3929/nextjs-setting (github.com)](https://github.com/bloodmoon3929/nextjs-setting) 해당 페이지와 똑같이 세팅
5. Tailwind css 설치
	`npm install -D tailwindcss`
	`npx tailwindcss init`
6. `npm run dev`로 실행 되는지 확인
