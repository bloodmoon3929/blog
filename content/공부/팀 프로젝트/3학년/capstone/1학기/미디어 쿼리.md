모바일 퍼스트(최근 선호되는 추세)
```css
/* Default CSS */
/* Mobile - Portrait */
/* Mobile - Landscape */
@media (min-width: 576px) {}

/* Tablet */
@media (min-width: 768px) {}

/* Desktop */
@media (min-width: 992px) {}

/* Desktop (Large) */
@media (min-width: 1200px) {}
```

```css
/* 작은 태블릿 */ 
@media (min-width: 576px) { 
	.container { 
		width: 90%; 
		min-height: 80vh; 
	} 
	.sign-in-container, .sign-up-container { 
		width: 50%; 
	} 
} 
/* 태블릿 */
@media (min-width: 768px) { 
	.container { 
		width: 85%; 
		min-height: 70vh; 
	} 
} 
/* 데스크탑 */ 
@media (min-width: 992px) { 
	.container { 
		width: 80%; 
		min-height: 60vh; 
	} 
} 
/* 큰 데스크탑 */ 
@media (min-width: 1200px) {

		min-height: 50vh; 
	}
}
```
